\subsection{Detailed Description}
The base class for any user processing class that wishes send \hyperlink{classmissive}{missive} s. 

A cell is the fundamental unit of concurrent computation of the library.

The user can think of it as a concurrent object (as in object oriented programming), an object that can receive \hyperlink{classmissive}{missive} s.
