The debug features are basically two\+:


\begin{DoxyItemize}
\item All functions in \hyperlink{log_8h}{log.\+h} to print and log in the host. Functions like\+: \hyperlink{log_8h_aae5500dae9c5dfaec69a65d9703dc50a}{mck\+\_\+slog} and \hyperlink{log_8h_a8ec0c02a2d94dec909d9cdc8340f733f}{mck\+\_\+slog2}. 
\item Functions to get the stack trace. When calling \hyperlink{trace_8h_a8090ce26e45526c1fd1947155a04398a}{mck\+\_\+get\+\_\+call\+\_\+stack\+\_\+trace} and when aborting execution by calling \hyperlink{global_8h_a6a6d1065bccc17077e1a6a0aeaa5f6dd}{mck\+\_\+abort} you get the stack trace printed in the host. 
\end{DoxyItemize}
