Introduction

This selected doxy generated documentation of the Messaging Cells Library is a description of the most releveant classes and functions that help in the understanding of the API and inner-workig of the library.

It is divided according to:

Debug features

The debug features are basically two:

  • You can log and print strings and numbers using functions like: mck_slog and mck_slog2. See log.h.
  • You can get the stack trace when calling mck_get_call_stack_trace and you can abort execution by calling mck_abort and get the stack trace printed in the host.
